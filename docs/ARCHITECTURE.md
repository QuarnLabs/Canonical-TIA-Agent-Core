# Frozen vs extensible layers

## Frozen

1. observation
2. compression to `z_t`
3. self-model update to `s_t`
4. loss and valence computation
5. structural measures `Phi_t`, `A_t`
6. interface synthesis `q_t`
7. phenomenal saturation proxy `chi_t`
8. policy selection `u_t`
9. environment update
10. parameter update

## Extensible

- neural architecture
- latent dimensionality
- specific estimators
- policy class
- environment
- perturbation adapters
